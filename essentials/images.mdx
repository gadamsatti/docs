## Images

### Using Markdown

Add images using markdown syntax:

```md
![EpicMarket Dashboard](/images/dashboard.jpg)
```

<Note>
  Image file size must be less than 5MB. For larger files, use a CDN like
  Cloudinary or AWS S3.
</Note>

### Using HTML

For more control over image display:

```html
<img height="400" src="/images/booking-flow.png" alt="EpicMarket Booking Flow"
style={{ borderRadius: '0.5rem' }} />
```

## Best Practices

1. **Optimize Images**

   - Compress images before uploading
   - Use appropriate file formats (PNG for screenshots, JPG for photos)
   - Provide descriptive alt text

2. **Responsive Design**

   - Use relative dimensions
   - Consider mobile viewers
   - Test across different screen sizes

3. **Loading Performance**
   - Lazy load images when possible
   - Use thumbnail previews for large images
   - Consider using a CDN for faster delivery

## Platform Configuration

Every EpicMarket integration needs proper configuration settings. Learn more about the available options below.

### Core Settings

<ResponseField name="businessName" type="string" required>
  The name of your business as it will appear on EpicMarket.

Example: `"Local Coffee Shop"`

</ResponseField>

<ResponseField name="apiKey" type="string" required>
  Your EpicMarket API key for authentication.
  <Note>Keep this key secure and never expose it in client-side code.</Note>
</ResponseField>

<ResponseField name="webhookUrl" type="string">
  URL where EpicMarket will send event notifications.

Example: `"https://your-domain.com/webhooks/epicmarket"`

</ResponseField>

### Display Options

<ResponseField name="theme" type="object">
  Customize the appearance of your EpicMarket integration
  <Expandable title="Theme Options">
    <ResponseField name="primaryColor" type="string">
      Main brand color in hex format
    </ResponseField>
    <ResponseField name="logo" type="string">
      URL to your business logo
    </ResponseField>
  </Expandable>
</ResponseField>

### Notification Settings

<ResponseField name="notifications" type="object">
  Configure how you receive updates
  <Expandable title="Notification Options">
    <ResponseField name="email" type="boolean" default={true}>
      Receive email notifications
    </ResponseField>
    <ResponseField name="sms" type="boolean" default={false}>
      Receive SMS notifications
    </ResponseField>
  </Expandable>
</ResponseField>

---
title: "Get Businesses"
openapi: "GET /businesses"
description: "Retrieve a list of businesses based on search criteria"
---

## Search Parameters

Use query parameters to filter and customize your business search:

- `category`: Filter by business type (e.g., "restaurant", "retail", "services")
- `location`: Search within a specific geographic area
- `limit`: Maximum number of results to return (default: 20)

## Response Format

The endpoint returns an array of business objects, each containing:

- Business profile information
- Location details
- Operating hours
- Category and tags

## Example Request

```bash
curl -X GET "https://api.epicmarket.com/v1/businesses?category=restaurant&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

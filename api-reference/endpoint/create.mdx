---
title: "Create Business"
openapi: "POST /businesses"
description: "Register a new business on EpicMarket"
---

## Request Body

Submit business details in the request body:

```json
{
  "name": "Local Coffee Shop",
  "description": "Artisanal coffee and pastries",
  "category": "cafe",
  "location": {
    "address": "123 Main St, City, State 12345",
    "coordinates": {
      "latitude": 37.7749,
      "longitude": -122.4194
    }
  },
  "operatingHours": {
    "monday": "7:00-19:00",
    "tuesday": "7:00-19:00",
    "wednesday": "7:00-19:00",
    "thursday": "7:00-19:00",
    "friday": "7:00-20:00",
    "saturday": "8:00-20:00",
    "sunday": "8:00-18:00"
  }
}
```

## Response

```json
{
  "id": "bus_12345",
  "name": "Local Coffee Shop",
  "status": "active",
  "created": "2024-01-20T08:00:00Z"
}
```

---
title: "Delete Business"
openapi: "DELETE /businesses/{id}"
description: "Remove a business from EpicMarket"
---

## Path Parameters

- `id`: The unique identifier of the business to delete

## Example Request

```bash
curl -X DELETE "https://api.epicmarket.com/v1/businesses/bus_12345" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

```json
{
  "id": "bus_12345",
  "status": "deleted",
  "deleted": "2024-01-20T15:00:00Z"
}
```

<Note>
  This action cannot be undone. Make sure to backup any important business data
  before deletion.
</Note>

## Welcome to EpicMarket API

Our API enables you to integrate EpicMarket's booking and discovery features into your applications. Whether you're a business looking to manage bookings or a developer building on top of our platform, our API provides the tools you need.

### Authentication

All API endpoints are authenticated using Bearer tokens. You'll need to register your application to receive API credentials.

  title="Get Your API Keys"

>

</Card>

### Core Endpoints

Our API provides endpoints for:

- Business Management
- Booking Services
- Order Processing
- Customer Management
- Analytics
